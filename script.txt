# ScriptGen/|y/;
a simple script generator

# Info/|n/;
a tool to help make scripts from parent script files
this tool splits scripts into different toggleable segments

Here are all of the escape sequences
/:h  : The Main Splitter;  this splits the code segmants
/.:/:.|y  : Enable By Default 
/.:/:.|n  : Disable By Default
/.:/:.;  : Mark Code 
/:.:/:.h  : gets replaced with /:h
.:/.:/:.:.  :  disables escape sequences when put in them
